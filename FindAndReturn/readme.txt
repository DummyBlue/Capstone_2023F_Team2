How to RUN

다음 두 개의 폴더를 생성해야 합니다.
/sounddata
/testdata

/sounddata 폴더 내부는 다음으로 구성되어야 합니다.
  /consonant
  /vowel

  위 두개의 폴더에 key_a_sliced와 같은 폴더들을 생성하고, 해당 폴더 안에 잘린 키 학습 데이터를 넣습니다.

/testdata 폴더 내부는 다음으로 구성되어야 합니다. (임시)
  /turtle

  해당 폴더 안에 슬라이스 된 추론하고자 하는 키 입력 값을 넣습니다.

잘 모르겠으면, zip파일을 그대로 풀어서 사용하시면 됩니다.
